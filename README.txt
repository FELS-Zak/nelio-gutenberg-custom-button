== Installation ==

This plugin can be installed manually.

1. Download the plugin.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Nelio Gutenberg Custom Button in the list and click the *Activate* link.

== Frequently Asked Questions ==

= Do I need Gutenberg to use Nelio Gutenberg Custom Button? =

Yes, you will need to use the block editor that comes with WordPress 5.0 or alternatively install the [Gutenberg plugin](https://wordpress.org/plugins/gutenberg/) to use Nelio Blocks.
